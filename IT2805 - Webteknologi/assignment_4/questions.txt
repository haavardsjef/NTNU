There are different advantages to having a single responsive website over dedicated mobile websites:
- All the functionality will stay the same across all sizes.
- Responsive sites can support a lot of different devices and screen sizes
- Makes updating the site easier, as you only have to update and maintain a single page.
- Search enginges might not always recognize that you want to access the mobile site, this is not a problem with responsive sites.
